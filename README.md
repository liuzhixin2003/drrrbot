# DrrrBot

NodeJS Bot for drrr.com

```
Usage: node main.js [OPTION]... [LambdaScript]...

OPTION:

-h, --help                       Show the manual

-i, --interact, --repl           Enable repl mode



$ # Run it interatively
$ node main.js

$ # Run the script
$ node main.js script/werewolf.js

```
